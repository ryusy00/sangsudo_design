시안 1 – 기본 리뉴얼

보기 ↗

요약

  • 구조 변경 없이 화면만 현대적으로 개선
  • 주요 서비스 아이콘 전면 배치로 빠른 접근
  • 배너·공지 한 영역에 모아 가독성 개선
  • 밝은 색상으로 안정적·친근한 인상

주요 영역

  • 메인 슬라이드 배너
  • 탭형 소식/공지 게시판
  • 자주 찾는 서비스 아이콘 8개
  • 수질 현황(기본 정보 중심)
  • 대표전화 간단 표기
기존 구조 유지 + 디자인 정리 중심

시안 2 – 확장형

보기 ↗

요약

  • Mega GNB로 상단에서 전체 메뉴 한눈에 확인
  • 좌측 배너·우측 소식카드로 주요 정보 집중 노출
  • 자주 찾는 서비스에 파스텔 색상으로 서비스 구분
  • 수질 정보(pH·색도·탁도·잔류염소)를 수치 카드로 분리
  • 푸른 밴드형 고객센터 영역으로 대표전화 강조

홍보관 영역

  • 메인 하단에 ‘상수도 홍보·체험관’ 카드 3개를 별도 영역으로 배치
  • 수돗물 체험캠프·정수장 견학·시민참여단 등 실제 운영 프로그램을 한눈에 소개
  • 각 카드에서 상세 안내·온라인 신청 페이지로 직접 연결
  • 요금·민원 중심 이미지를 넘어 체험·교육·소통하는 기관 이미지로 확장
시민참여를 늘리고 기관 브랜드 이미지를 높이는 고도화 제안
